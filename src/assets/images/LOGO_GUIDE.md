# Mahalakshmi Granites Logo Guide

## Logo Variations

### 1. **logo.svg** - Circular Badge
- **Use Case**: App icons, social media avatars, mobile apps
- **Dimensions**: 200x200px
- **Background**: Dark gradient with gold accents
- **Best For**: Square spaces, profile pictures

### 2. **logo-horizontal.svg** - Horizontal Layout
- **Use Case**: Website header, email signatures, letterheads
- **Dimensions**: 400x100px
- **Background**: Transparent
- **Best For**: Light backgrounds, website navigation

### 3. **logo-white.svg** - White Text Version
- **Use Case**: Dark backgrounds, footers, promotional materials
- **Dimensions**: 400x100px
- **Background**: Transparent with white text
- **Best For**: Dark backgrounds, hero sections

## Color Palette

## Design Elements

### Diamond Symbol
The diamond represents:
- **Premium Quality**: Like polished granite
- **Durability**: Strength and longevance
- **Elegance**: Natural beauty and sophistication
- **Precision**: Expert craftsmanship

### Typography
- **Primary Font**: Georgia (Serif) - Traditional elegance
- **Secondary Font**: Arial (Sans-serif) - Modern clarity
- **Letter Spacing**: Wide spacing for premium feel

## Usage Guidelines

### ✅ DO:
- Maintain clear space around the logo (minimum 20px)
- Use appropriate version for background color
- Scale proportionally
- Export in SVG for web use
- Use high-resolution PNG for print

### ❌ DON'T:
- Stretch or distort the logo
- Change the colors without approval
- Add effects or filters
- Place on busy backgrounds
- Use low-resolution versions

## File Formats

- **SVG**: Vector format for web (scalable, small file size)
- **PNG**: Raster format for print and presentations
- **ICO**: Favicon for browser tabs
